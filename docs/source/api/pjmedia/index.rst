PJMEDIA
============================================


PJMEDIA is a fully featured open source media stack, featuring small footprint and 
good extensibility and excellent portability.

PJMEDIA contains the following libraries:

.. list-table::
   :header-rows: 0

   * - PJMEDIA
     - The base media framework and contains all main media components
   * - PJMEDIA Codec
     - Collection of audio and video codec implementations.
   * - PJMEDIA Audio Device
     - Audio device framework and abstraction library for native audio device.
   * - PJMEDIA Video Device
     - Video device framework and abstraction library for native video device.
